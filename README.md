# Health Application

## _Based on ReactJS & Vite_

Library used:

- Mock Service Worker
- React Query
- Axios
- React Hook Form
- Recharts
- React Router Dom
- Lodash, Date-fns, yup...

## Features

- Home Page (My analysis, My Meal)
- My Body Page (My exercise, My records, My diary)
- Column Page (Recommend, Posts)

## Envinroment

```sh
VITE_API_URI="http://localhost:3000"
```

## Installation

```sh
yarn
```

```sh
yarn dev
```

```sh
127.0.0.1:3000
```

## Build Production

```sh
yarn build
```
